\doxysubsection{Detailed Description}
Estrutura que representa um elemento volumétrico (voxel). 

Um voxel é um pequeno elemento tridimensional, usado para criar figuras em um espaço 3D. Esta estrutura é usada para representar os atributos de um voxel, incluindo cor (r, g, b, a) e se ele deve ser exibido (show).
